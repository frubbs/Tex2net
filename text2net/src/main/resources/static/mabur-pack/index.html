The solution, Step by Step

The basic engine is the same for all tools. The difference between them is just the input.

Define Input

Select the text input. Check "tools" for options.

Define node pattern

What should be considered a node in the generated network? A single word (prepositions discarded)? A person name? (We can detect most Brazilian names with family name). Write your own rule if you wish.

Define limits (optional)

Tags, words, elements that define subgroups of text which determine if elements are connected. Ex: Connect only elements found in the same Chapter, section, page, between words "Hello" and "Bye", etc...

Generate network

Relax and let us do the work. Outputs are 1-mode networks in pajek format. You can merge multiple inputs into one network.
